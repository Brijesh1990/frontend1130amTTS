What is External Javascript ?

javascript is saved as external file using .js. and called in html body or head tag using script tag i.e called external javascript

examples : make a javascript external file with name of name.js

name.js

document.write("My name is Aditi");

examples :call inside of html web page

name.html

         

< !DOCTYPE html>
< html>
< head>
    < meta charset='utf-8'>

    < title>Page Title
    < meta name='viewport' content='width=device-width, initial-scale=1'>
    < link rel='stylesheet' type='text/css' media='screen' href='main.css'>

< /head>
